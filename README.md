# 异环 NTE / Roblox 反作弊冲突一键切换工具

这是一个 Windows 批处理脚本，用来缓解《异环》（英文名：NTE）反作弊驱动与 Roblox 不能共存的问题。

如果你的电脑安装过《异环》，即使没有启动《异环》，重启后 Roblox 仍然可能无法正常运行。本工具通过按需启用或禁用《异环》的反作弊驱动，让你可以在同一台电脑上切换游玩 Roblox 和《异环》。

## 它解决什么问题

- 安装《异环》后，不运行《异环》也无法启动 Roblox。
- 每次手动打开注册表修改驱动启动项很麻烦。
- 想玩 Roblox 时禁用《异环》反作弊驱动，想玩《异环》时再恢复启用。

## 使用方法

1. 下载本仓库中的 `NTE_Roblox.bat`。
2. 双击运行脚本。
3. Windows 弹出管理员权限确认时，选择“是”。
4. 根据你现在想玩的游戏选择：
   - 输入 `1`：禁用《异环》反作弊驱动，之后可以去玩 Roblox。
   - 输入 `2`：启用《异环》反作弊驱动，之后可以去玩《异环》。
   - 输入 `3`：退出脚本。
5. 按脚本提示重启电脑。

注意：修改反作弊驱动状态后，必须重启电脑才会完全生效。

## 工作原理

脚本会修改 Windows 注册表中的这个服务项：

```text
HKLM\SYSTEM\CurrentControlSet\Services\HtAntiCheatDriver
```

具体修改的是 `Start` 值：

| 操作 | Start 值 | 含义 |
| --- | --- | --- |
| 禁用《异环》反作弊驱动 | `4` | Disabled |
| 启用《异环》反作弊驱动 | `2` | Automatic |

脚本只是把这个修改过程自动化，避免每次手动打开注册表编辑器。

## 常见问题

### 为什么需要管理员权限？

因为脚本需要修改 `HKEY_LOCAL_MACHINE` 下的系统服务注册表项。普通用户权限无法修改这个位置。

### 为什么修改后还要重启？

反作弊驱动属于系统级驱动，启动状态通常需要在系统启动阶段加载或停用。只修改注册表不重启，状态可能不会立刻生效。

### 选择“禁用”后还能玩《异环》吗？

通常不能。禁用反作弊驱动后，《异环》可能无法正常启动或通过反作弊检查。想玩《异环》时，请重新运行脚本并选择 `2` 启用，然后重启电脑。

### 选择“启用”后 Roblox 又不能玩怎么办？

如果 Roblox 再次受到影响，请重新运行脚本，选择 `1` 禁用《异环》反作弊驱动，然后重启电脑。
